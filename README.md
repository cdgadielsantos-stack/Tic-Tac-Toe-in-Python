# 🎮 Jogo da Velha em Python

Um projeto simples de **Jogo da Velha (Tic-Tac-Toe)** desenvolvido em Python para rodar no terminal.

## 📌 Sobre o projeto

Este projeto foi criado com o objetivo de praticar lógica de programação utilizando Python, implementando um dos jogos mais clássicos: o Jogo da Velha.
